🧩 Quitar descarga forzada en EVA FING

Extensión de Chrome que desactiva la descarga automática de archivos en EVA FING (Moodle), permitiéndote abrir PDFs y otros archivos directamente en el navegador sin tener que descargarlos.

🧠 Ideal para estudiantes que quieren previsualizar apuntes, parciales o materiales sin llenar la carpeta de “Descargas” de cosas que abren una sola vez.

✨ ¿Qué hace?

Convierte automáticamente todos los enlaces con forcedownload=1 en forcedownload=0.

🧪 Ejemplo

Antes:

https://eva.fing.edu.uy/pluginfile.php/123/mod_resource/content/1/archivo.pdf?forcedownload=1

Después:

https://eva.fing.edu.uy/pluginfile.php/123/mod_resource/content/1/archivo.pdf?forcedownload=0

✅ ¡Ahora el navegador lo abre directamente!

🧑‍💻 ¿Cómo instalar en Google Chrome?

  1. Descargá este repositorio: 
    git clone https://github.com/nicoserra03/no-forcedownload-eva-extension.git
    o usá el botón Code ▾ → Download ZIP.
  2. Abrí chrome://extensions/ en el navegador
  3. Activá el Modo desarrollador (arriba a la derecha)
  4. Hacé clic en “Cargar descomprimida”
  5. Seleccioná la carpeta del proyecto
  6. ¡Listo! 🎉 EVA Fing ya no te va a forzar descargas innecesarias.
